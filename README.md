# HK-SG Hackathon Algo Trading Bot

This is a repo for an algo trading bot competing in the [SG vs. HK Quant Trading Hackathon (Universities)](https://luma.com/tqx5xvcy?tk=Laf5YB) by team UnitSetGo.

The system is designed around the competition requirement:
 - Ran on AWS EC2 instance, fully automated
 - Interface to the Roostoo API for trading crypto
 - Mid-frequency trading strategies with 1-min trading interval at the highest

 The system is currently written in Python 3.12, and SQLite as the database.
